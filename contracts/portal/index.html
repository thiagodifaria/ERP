ERP Integration Portal

Central contract and API navigation surface

This portal is the central entrypoint for HTTP contracts, event schemas and governance artifacts of the ERP platform. It is intentionally static and versioned with the monorepo, so contract discovery stays aligned with the same commit that shipped the code.
