# service-typescript

TypeScript is reserved for product control plane services and external product integrations.

Planned services:

- workflow-control
- engagement
- shared

Standard layout:

- `src/server.ts`
- `src/api`
- `src/application`
- `src/domain`
- `src/infrastructure`
- `src/providers`
- `src/jobs`
- `src/config`
- `src/telemetry`
- `tests/unit`
- `tests/integration`
- `tests/contract`
